AI Agent Assets

- `GenerationPrompt.txt`: primary instruction prompt used for agent-driven generation.
- `context_examples/`: curated NL + SysML grammar exemplars consumed by the agent.
- `generation_prompts/`: NL-only prompt inputs.
- `Generated_from_Prompts_AI_AGENT/`: generated SysML outputs and evaluation artifacts.
- `analysis/`: consolidated analysis artifacts (summary text and result JSON files).
- `timings.csv`: prompt-by-prompt generation timing log.
- `formal-25-09-05.pdf`: SysML language reference used during generation.

This folder groups the agent-mode workflow assets.
